Performance tips in Julia
  1. Performance critical code should be inside a function
  2. Avoid untyped global variables
  3. Pay attention to memory allocation
  4. Avoid containers with abstract type parameters
  5. Avoid struct fields with abstract type
  6. Break functions into multiple definitions
  7. Write “type-stable” functions
  8. Avoid changing the type of a variable
  9. Access arrays in memory order, along columns
  10. Pre-allocating outputs
  11. Fuse vectorized operations
  12. Consider using views for slices
  13. Copy irregularly-accessed data into a contiguous array before operating on it
  14. Consider StaticArrays.jl for small fixed-size vector/matrix operations
  15. Avoid string interpolation for I/O
  16. Performance Annotations: @fastmath @inbounds @simd
